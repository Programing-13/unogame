# UNO

우노(UNO) 게임은 세계에 가장 많이 알려져 있는 카드 게임 중 하나이다. 
상대방(COMPUTER)과 나(ME)는 각각 7장씩 카드를 받는다. 중간에 있는 카드와 숫자가 같거나 색깔이 같으면 카드를 낼 수 있다.
카드를 한번에 몇개를 내는지는 상관이 없다. 만약 1개가 남았을때 UNO! 를 지정된 시간안에 외치지 않으면 카드를 낼 수 없고 한장 추가해야한다.
ENJOY YOUR GAME!
